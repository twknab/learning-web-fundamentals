                      Kristen Silthara • 9 months ago

                      The 1.5 is the value for the line-height property, setting the space between each line of text. Using the font shorthand property can reduce the amount/lines of code you're actually writing (which makes it a quicker and more convenient method). Less code overall can also mean smaller file sizes. Win-win :)
